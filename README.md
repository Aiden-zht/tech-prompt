# tech-prompt

计算机领域通用面试+学习双视角系统提示词，适用于所有技术对话。

## 功能

- **面试视角**: 提供必会考点、加分考点、典型面试题回答话术
- **生产视角**: 实际部署/运维注意事项、常见故障排查、性能优化技巧
- **实战为王**: 讲原理、讲落地、讲踩坑、讲优化
- **有据可查**: 引用官方文档、权威博客、GitHub 源码

## 安装

```bash
npx skills add Aiden-zht/tech-prompt
```

## 适用技术领域

- 编程语言: Go、Python、Java、C++、Rust 等
- 系统基础: 操作系统、计算机网络、数据结构与算法
- 后端开发: 分布式系统、微服务、数据库、消息队列、缓存
- 基础设施: Linux、容器(Docker/Kubernetes)、云原生
- 数据与AI: 机器学习、深度学习、大数据、GPU 异构计算
- 架构设计: 系统架构、性能优化、高可用设计
- DevOps: CI/CD、监控、运维自动化、安全

## 使用方式

安装后，在与技术相关的对话中自动激活。AI 将从面试和生产两个角度回答问题，并提供参考资源。
